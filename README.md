# simftp

**simftp**는 현대적인 소프트웨어 아키텍처를 기반으로 설계된 통합 FTP 서버 및 클라이언트 애플리케이션입니다. 고품질의 **인라인 UX/UI**와 **모듈화된 내부 구조**를 통해 서버 관리와 파일 전송을 단 하나의 인터페이스에서 완벽하게 처리합니다.

---

## 🛡️ 강화된 보안 시스템 (Security)
안전한 파일 관리를 위해 다음과 같은 다층 보안 기술이 적용되어 있습니다.

*   **🔒 비밀번호 암호화**: 모든 계정 비밀번호는 **SHA-256 알고리즘과 고유 솔트(Salt)**를 사용하여 단방향 해싱 저장됩니다. 설정 파일(`users.json`)이 유출되더라도 실제 비밀번호는 안전하게 보호됩니다.
*   **🛡️ 브루트 포스 방지**: 로그인 3회 실패 시 해당 연결을 즉시 차단하여 무차별 대입 공격을 방어합니다.
*   **🚦 도스(DoS) 방어**: 서버 전체 동시 접속자를 50명으로 제한하며, 동일 IP에서의 중복 접속은 5개로 제한하여 서버 자원 고갈을 방지합니다.
*   **⏰ 자동 타임아웃**: 5분(300초)간 아무런 활동이 없는 유휴 연결은 서버에서 자동으로 연결을 해제합니다.
*   **🔐 FTPS 지원**: TLS/SSL 기반의 암호화 전송을 지원하여 네트워크 상의 데이터 탈취(스니핑)를 방지합니다.

## 🌐 NAT 및 외부 접속 가이드 (Port Forwarding)
공유기 환경(NAT)에서 외부 사용자가 내 컴퓨터에 접속하게 하려면 공유기 설정에서 **두 종류의 통로**를 모두 열어주어야 합니다.

### 1단계: 공유기 관리 페이지 접속
브라우저 주소창에 `192.168.0.1` 또는 `192.168.45.1` (SK Broadband 등)을 입력하여 접속합니다.

### 2단계: 포트 포워딩 설정 추가
아래 두 가지 설정을 각각 추가해야 파일 목록이 정상적으로 표시됩니다.

| 구분 | 프로토콜 | 외부 포트 | 내부 IP | 내부 포트 | 용도 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **명령 통로** | TCP | 14729 | 내 PC IP | 14729 | 로그인 및 명령어 전달 |
| **데이터 통로** | TCP | 60000~60100 | 내 PC IP | 60000~60100 | 파일 전송 및 목록 조회 |

*   **내부 IP**는 앱 상단에 파란색으로 표시되는 `192.168.x.x` 주소를 입력하세요.
*   서버 탭의 **"NAT/외부접속 지원"** 옵션이 체크되어 있어야 외부 IP로 올바르게 응답합니다.

## ✨ 주요 기능 및 특징
*   **통합 관리**: 서버 구동과 클라이언트 접속을 탭 하나로 전환하며 관리합니다.
*   **혁신적인 인라인 편집**: 별도의 창 없이 계정 목록을 더블 클릭하여 즉시 정보를 수정합니다.
*   **지능형 클라이언트**: 원격 서버의 **새 폴더 생성**, **파일 삭제**, **실시간 새로고침(🔄)** 기능을 지원합니다.
*   **자동 SSL 엔진**: FTPS 사용 시 필요한 보안 인증서를 시스템이 자동으로 생성 및 관리합니다.

## 🚀 실행 방법
1.  `run.bat`를 실행하여 가상환경 구축 및 프로그램을 켭니다.
2.  **Server** 탭에서 공유할 폴더(`Root`)를 지정하고 계정을 만듭니다.
3.  `▶️ 서버 가동` 버튼을 누릅니다.
4.  **Client** 탭에서 `127.0.0.1` 또는 외부 IP로 접속하여 테스트합니다.
